# show-copy
> Show unsplash api copyright infos.

## Install
```sh
	$ npm install @splash-cli/show-copy --save

	#or

	$ yarn add @splash-cli/show-copy
```

## Usage
```js
	import showCopy from '@splash-cli/show-copy';

	showCopy(photoData, info) //=> prints infos
```
